Projects

Partners in Health

REACT Firebase Stripe Axios HTML5 CSS3 Responsive

Inspired by a novel written about the work of the Partners in Health organization, I created a multi-page application that highlights the projects and aspirations of the organization. This website was built using React and makes use of React-router to ensure that pages load quickly on change. The website is fully responsive and includes a functional donation modal that is integrated with the Stripe payment API. There is also a contact page which utilizes form validation before pushing the contact information to a Firestore Database.

Urban Touch

REACT Firebase Stripe Axios HTML5 CSS3 Responsive

This e-Commerce prototype has been built from the ground up and features a fully functioning cart and payment system using the Stripe API. I utilized Google’s Firebase Technology to set up a user authentication system as well as a Firestore database that allows users to create an account and view their previous orders. The website is fully responsive and uses an original minimal design in order to make navigation of the website as simple as possible.

Weather Application

JavaScript Responsive API HTML5 CSS3

This application uses the OpenWeatherMap API to display global weather information to users. The user can input any city in the world into the search bar and the daily weather information as well as the five day forecast will be fetched and displayed to the user. This website is fully responsive and the weather icons in the forecast section will update as well to represent the type of weather expected for that day. This project is written in vanilla JavaScript and utilizes DOM manipulation techniques to update the visual display.
